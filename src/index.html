Projects

FaceTrace

FaceTrace is a web app I built to practice full-stack fundamentals. It lets users input image URLs and automatically detects and traces the faces within them. Built with React.js, Node.js, and PostgreSQL, and supported by libraries like bcrypt, cors, knex, and tachyons, this project was both a fun challenge and a great learning experience. Try it out yourself — see how many faces you can trace!

Book Buddy

This was a project to showcase my ability to use React. It is a functional client I built for the Book Buddy API. The app connects to a catalog of books where logged-in users can reserve or return titles.
